## Run Locally

Clone the project

```bash
  git clone https://github.com/ManishSheela/Airblack-Tech-Assignment.git
```

Go to the project directory

```bash
  cd airblack
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

**Client:** React, CSS
